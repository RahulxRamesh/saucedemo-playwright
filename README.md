# saucedemo-playwright

###Install
- git clone https://github.com/RahulxRamesh/saucedemo-playwright.git
- npm install
- npm init playwright@latest

###Run tests
- npx playwright test
- npm run test
- npm run test:chrome

    ##View run results
    - npm run reports

###Base playwright config
- Runs each test on Chrome, Firefox, and Webkit (safari)
- Runs /tests folder
